# Handwriting Portfolio

A beautiful landing page to showcase handwriting work, calligraphy, and decorative lettering projects.

## 🎨 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and elegant interface with smooth animations
- **Gallery Section**: Showcase multiple handwriting samples
- **Smooth Navigation**: Easy-to-use navigation with smooth scrolling
- **Professional Layout**: Organized sections for portfolio presentation

## 📁 Project Structure

```
handwritting-5/
├── index.html       # Main landing page
├── styles.css       # Styling and responsive design
├── script.js        # Interactive functionality
└── README.md        # Project documentation
```

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Shubhambisht212/handwritting-5.git
```

2. Navigate to the project directory:
```bash
cd handwritting-5
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

4. Visit `http://localhost:8000` in your browser

## 🎯 Customization

### Adding Gallery Images

Replace the placeholder gallery items in `index.html` with your actual images:

```html
<div class="gallery-item">
    <img src="path/to/your/image.jpg" alt="Your handwriting sample">
    <p class="gallery-title">Your Title Here</p>
</div>
```

### Updating Colors

Modify the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #e74c3c;
    --accent-color: #3498db;
    /* ... */
}
```

### Adding Contact Information

Update the contact section in `index.html` with your email or social media links.
